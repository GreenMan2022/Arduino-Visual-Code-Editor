## 🚀 Возможности

### ✨ Визуальное редактирование
- **Построчный просмотр** — каждая строка кода отображается отдельно с номерами
- **Интерактивные поля** — кликай на любой элемент и меняй его значение
- **Авто-расширение** — поля подстраиваются под длину текста
- **Горизонтальная прокрутка** — длинные строки не обрезаются

### 🎯 Что можно редактировать

| Элемент | Что можно менять | Тип контрола |
|---------|------------------|--------------|
| **Типы переменных** | int, float, byte, char, long, unsigned int, const int, bool, String | Выпадающий список |
| **Имена переменных** | Любое имя | Текстовое поле |
| **Значения переменных** | Числа, пины, строки | Поле ввода |
| **Пины** | Любой номер (0-255, A0-A15) | Текстовое поле |
| **Режимы** | OUTPUT/INPUT/INPUT_PULLUP | Выпадающий список |
| **Состояния** | HIGH/LOW | Выпадающий список |
| **Задержки** | Любое выражение (delayTime, delayTime/2, 500) | Текстовое поле |
| **Условия** | Любое выражение для if/while | Текстовое поле |
| **Циклы for** | init, condition, increment | Текстовые поля |
| **Строки** | Текст в кавычках | Текстовое поле |

### 🧠 Умные функции
- **Авто-определение типа** — числа, пины, строки распознаются автоматически
- **Подсказки для пинов** — выпадающий список A0-A15 для аналоговых пинов
- **Сохранение отступов** — код сохраняет свою структуру
- **Мгновенное обновление** — изменения сразу применяются к коду
